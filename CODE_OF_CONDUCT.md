# Code of Conduct

We are committed to a harassment-free, welcoming project for everyone, regardless of age, body size, disability, ethnicity, sex characteristics, gender identity and expression, experience level, education, socioeconomic status, nationality, personal appearance, race, caste, religion, or sexual identity and orientation.

## Expected behavior

Be respectful, assume good intent while evaluating evidence, accept constructive feedback, focus on what is best for users and maintainers, and protect private information. Disagreement about code or risk is welcome; personal attacks are not.

## Unacceptable behavior

Harassment, discrimination, sexualized attention, threats, doxxing, publishing private information, sustained disruption, metric manipulation, spam contributions, and knowingly misleading technical claims are unacceptable.

## Enforcement

Project maintainers may edit, reject, or remove contributions and restrict participation when behavior violates this policy. After publication, report conduct concerns privately through the contact method listed on the maintainer's GitHub profile. Do not use a public issue for a sensitive report.

This policy is inspired by Contributor Covenant 2.1 and applies in project spaces and when representing the project publicly.
